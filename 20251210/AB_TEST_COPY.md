# 🧪 로또인사이트 A/B 테스트용 카피 12종 패키지

한국 시장 전환율 최적화를 위한 테스트용 카피 세트입니다.
각 타입별로 3종씩, 총 12종의 카피를 제공합니다.

---

## 📊 A/B 테스트 가이드

### 테스트 우선순위
1. **Hero 헤드라인** (가장 큰 영향)
2. **CTA 버튼 문구** (직접적 전환)
3. **무료 제공 섹션** (가입 유도)
4. **서브 헤드라인** (신뢰/호기심)

### 테스트 기간
- 최소 7일 / 방문자 1,000명 이상 권장
- 전환율 차이 5% 이상 시 통계적 유의미

---

## 🏆 1. Hero 헤드라인 (4종)

### A타입 - 신뢰형 (Trust)
```
"10년치 데이터 기반 AI 분석
이번 주 조합을 확인하세요"
```
**타겟:** 신중한 사용자, 30대 이상
**포인트:** 데이터, 검증, 안정감

---

### B타입 - 호기심형 (Curiosity)
```
"이번 주 데이터에서
특이한 패턴이 감지되었습니다"
```
**타겟:** 호기심 많은 사용자
**포인트:** 궁금증 유발, 클릭 유도

---

### C타입 - 감정 자극형 (Emotion)
```
"놓치지 마세요
지금이 가장 유리한 시점입니다"
```
**타겟:** FOMO 민감 사용자
**포인트:** 긴급성, 기회 상실 두려움

---

### D타입 - 즉시성형 (Instant)
```
"1초 후
당신의 조합이 공개됩니다"
```
**타겟:** 빠른 결과를 원하는 사용자
**포인트:** 즉각적 보상, 속도감

---

## 🔘 2. CTA 버튼 문구 (4종)

### A타입 - 무료 강조
```
무료 2회로 바로 확인 →
```
**예상 CTR:** 높음 (무료 = 진입 장벽 낮춤)

---

### B타입 - AI 강조
```
AI가 뽑은 번호 보기 →
```
**예상 CTR:** 중상 (AI 신비감 활용)

---

### C타입 - 이번 주 강조
```
이번 주 추천 조합 보기 →
```
**예상 CTR:** 중상 (시의성 강조)

---

### D타입 - 개인화 강조
```
나만의 행운 조합 생성하기 →
```
**예상 CTR:** 중 (개인화 느낌)

---

## 🎁 3. 무료 제공 섹션 헤드라인 (4종)

### A타입 - 직접적
```
회원가입 시 즉시 2회 무료 분석!
```
**특징:** 명확한 혜택 전달

---

### B타입 - 한정 강조
```
지금 가입하면 무료 2회 제공 — 한정 이벤트
```
**특징:** FOMO + 긴급성

---

### C타입 - 가치 강조
```
무료로 AI 분석 경험해보세요
가입 즉시 2회 제공
```
**특징:** 경험 가치 강조

---

### D타입 - 패턴 연결
```
이번 주 패턴이 달라졌습니다
무료 2회로 직접 확인해보세요
```
**특징:** 호기심 + 무료 혜택 결합

---

```
AI가 10년치 데이터를 분석합니다."
```

---

### D타입 - 사용자 검증
```

검증된 분석 시스템."
```
